# SaS2.Save.Editor

Salt and Sacrifice v2.0.0.1 save editor desktop application

## Features
- Edit player name
- Edit Silver value
- Edit Experience value
- Edit Haze burnt state
- Edit item count
- Edit item upgrade
- Add items
- See player stats
- See player flags

## Requirements

- Windows
- .NET 8.0

##  Development

### Requirements
- .NET 8.0
- Visual Studio 2019+

- Checkout [SaS2.Save.Editor](https://github.com/spodlesniy/SaS2.Save.Editor) project
- Checkout [SaS2.Save](https://github.com/spodlesniy/SaS2.Save) project. Put it in parent folder
